
# MLB Hot Hand Analysis

Statistical analysis of the hot hand phenomenon in Major League Baseball


## Project Overview

This project investigates the existence of the hot hand phenomenon in Major League Baseball (MLB).

The goal is to determine whether a player's recent success influences future performance or whether observed streaks can be explained by random variation.

## Research Questions

- Does the hot hand exist in MLB?
- Are hitting streaks statistically significant?
- How does observed performance compare to random expectations?
